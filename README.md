# 🐞 Funny Insect Animation

**An Interactive Mouse-Following Insect Built with HTML, CSS, JavaScript & D3.js**

🐞 Animated insect • 🎯 Mouse tracking • 🦵 Dynamic leg movement • ⚡ SVG powered interactions

Funny Insect Animation is a creative frontend project that brings a robotic insect to life. The insect intelligently follows the user's cursor while dynamically animating its legs and body segments, creating a smooth crawling effect.

Designed for learning **SVG animation, D3.js, motion algorithms, interactive graphics, and creative frontend development**.

---

# ✨ Features

🐞 **Interactive Insect Movement**

The insect continuously follows the mouse cursor in real time.

🎯 **Mouse Tracking System**

Responsive movement based on cursor position.

🦵 **Animated Leg Motion**

Realistic crawling effect using dynamically updated SVG paths.

🔗 **Segment-Based Body Design**

Multiple connected body segments create smooth motion.

⚡ **Real-Time Rendering**

Continuous animation updates for fluid interaction.

🎨 **SVG Graphics**

Built entirely using scalable vector graphics.

📱 **Cross-Browser Compatible**

Works on all modern desktop browsers.
